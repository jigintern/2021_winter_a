# 切り抜きまとめサイト(仮)

> [2021年冬jig.jpインターンプログラム](https://jig.jp/topics/202012252000/) チームAの作品です！

## 使用の前に

YouTube Data APIのAPIキーが必要です。

https://console.developers.google.com/

取得し終わったら、ファイル `apikey.txt` を作成し、貼り付けてください。

## 使用方法

```sh
$ deno run -A server.js
```

http://localhost:8001

で動きを確認することができます。
